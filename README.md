# 📰 News Parser

Парсер новостей с красивым интерфейсом. Собирает новости по теме и показывает в удобном виде.

## Возможности

- Парсинг новостей с RSS-лент
- Фильтрация по ключевым словам
- Красивый интерфейс
- Экспорт в JSON
- Автообновление

## Быстрый старт

### 1. Установи зависимости

```bash
pip install -r requirements.txt
```

### 2. Запусти

```bash
python app.py
```

Открой http://localhost:5001 в браузере.

## Использование

1. Введи тему или ключевое слово
2. Нажми "Найти новости"
3. Просмотри результаты
4. Экспортируй при необходимости

## Стек

- Python 3.10+
- Flask
- feedparser
- requests
